# Playwright Automation Framework

This repository contains an automated end-to-end testing framework built for an e-commerce application (SauceDemo). It is designed to demonstrate scalable testing practices and CI/CD integration.

##  Technologies Used
* **Automation Tool:** Playwright
* **Language:** TypeScript
* **Design Pattern:** Page Object Model (POM)
* **CI/CD:** GitHub Actions

##  Scenarios Covered
* Data-driven login verification (Valid, Invalid, Locked-out users).

##  How to Run Locally
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npx playwright test` to execute the test suite.
4. Run `npx playwright show-report` to view the HTML test report.
